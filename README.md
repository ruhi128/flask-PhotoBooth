# Photobooth Filter Fix

## Installation

1. Create folder structure:
   ```
   photobooth/
   ├── app.py (use app_photobooth.py)
   ├── templates/
   │   └── photobooth.html
   └── static/
       └── photobooth.css
   ```

2. Run the application:
   ```bash
   python app.py
   ```

3. Open browser to `http://localhost:5000`
